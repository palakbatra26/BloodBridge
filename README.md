## About BloodBridge

BloodBridge is a comprehensive blood donation management platform that connects blood donors with recipients, hospitals, and blood banks. Our mission is to make blood donation easier, more accessible, and more impactful.

## üåü Key Features

### For Donors
- **Easy Registration**: Quick and simple donor registration process
- **Donation History**: Track your donations and impact
- **Rewards & Recognition**: Earn points and badges for donations
- **Urgent Alerts**: Get notified when your blood type is needed
- **Health Tracking**: Monitor your donation eligibility

### For Recipients & Families
- **Blood Requests**: Submit urgent blood requests
- **Real-time Matching**: Connect with compatible donors quickly
- **Hospital Network**: Access to verified hospitals and blood banks
- **24/7 Support**: Round-the-clock assistance

### For Hospitals & Blood Banks
- **Inventory Management**: Track blood supply efficiently
- **Camp Organization**: Organize and manage blood donation camps
- **Donor Database**: Access to verified donor network
- **Analytics**: Insights into donation patterns and needs

### Smart Features
- **AI Chatbot**: 24/7 assistance for all blood donation queries
- **GPS Location**: Find nearby blood camps and donation centers
- **Eligibility Checker**: Instant eligibility verification
- **Rating System**: Rate and review blood camps
- **Feedback System**: Share your donation experience

## üöÄ Quick Start

### Prerequisites
- Node.js (v14 or higher)
- MongoDB
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/bloodbridge.git
cd bloodbridge
```

2. **Install backend dependencies**
```bash
cd backend
npm install
```

3. **Install frontend dependencies**
```bash
cd ../red-connect-app
npm install
```

4. **Set up environment variables**

Create `.env` file in backend directory:
```env
MONGODB_URI=your_mongodb_connection_string
PORT=5002
CLERK_SECRET_KEY=your_clerk_secret_key
JWT_SECRET=your_jwt_secret
```

Create `.env` file in frontend directory:
```env
VITE_API_URL=http://localhost:5002
VITE_CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key
```

5. **Start the servers**

Backend:
```bash
cd backend
npm start
```

Frontend:
```bash
cd red-connect-app
npm run dev
```

6. **Access the application**
- Frontend: http://localhost:5173
- Backend API: http://localhost:5002
